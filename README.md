#  Alerta Vizinhança

Um aplicativo colaborativo que ajuda comunidades a se protegerem, permitindo que moradores enviem e recebam alertas em tempo real sobre situações suspeitas na vizinhança.

---

##  Sobre o Projeto

O **Alerta Vizinhança** foi criado com o objetivo de aumentar a segurança local, conectando vizinhos e tornando a comunicação sobre ocorrências mais rápida e eficiente.

Com ele, qualquer morador pode:
- Notificar sobre situações suspeitas;
- Receber alertas de outros usuários próximos;
- Compartilhar informações em tempo real com segurança.

---

##  Funcionalidades

-  Envio de alertas com localização
-  Notificações instantâneas
-  Comunicação entre vizinhos
-  Mapa interativo com ocorrências
-  Canal direto com autoridades locais (em versões futuras)

---

##  Tecnologias Utilizadas

- **Frontend:** React Native  
- **Backend:** Firebase / Node.js  
- **Mapas:** Google Maps API  
- **Autenticação:** Firebase Auth
